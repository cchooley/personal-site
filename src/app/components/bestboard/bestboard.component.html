BestBoard

Built using: React, React Router, Semantic UI, Stripe, Chart.js, PostgreSQL, bcrypt, JWT, Knex.js, Express.js, Node.js, Heroku, Firebase

BestBoard is my captstone project for the WDI program at Galvanize. It is an app that was conceived as a means of improving productivity, communication, and transparency for 501(c)3 Boards of Directors.

Per the WDI curriculum, this was an app built in less than two weeks and prominently featuring a technology that I had not used before. My unfamiliar technology here was React.js (and corresponding React libraries), which I had never personally used prior to this project.

BestBoard's current iteration has features such as account creation (protected with auth via bCrypt and JWT), voting management, static payment processing with Stripe, a calendar via React Big Calendar, data visualization with Chart.js, admin features and full CRUD functionality.

Still needed: I believe this app needs dynamic payment processing, task management tracking and document uploading to be truly complete. But who am I kidding? That's probably barely scratching the surface.
